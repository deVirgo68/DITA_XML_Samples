What is Multiplication?

"Multiplication" is a mathematical operation. It is an action that is performed using 2 or more numbers called factors. The result is called the product.

Multiplication is repeated addition. 3 + 3 + 3 = 9 in multiplication is represented as 3 x 3 = 9. Each number 3 is a factor and the result, or product is 9.

Use the multiplication (x) button to express the multiplication operation in the JPS Calculator app.

With the multiplication process, you take 2 numbers and increase it to a larger number.
